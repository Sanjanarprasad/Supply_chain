🌱 AgroLedger

Farmer Provenance System for Agricultural Supply Chain


📌 Overview

AgroLedger is a simple and reliable system designed to improve transparency and trust in agriculture. In traditional supply chains, buyers often rely on verbal claims about product quality and origin. AgroLedger solves this by giving every crop batch a digital identity that can be verified instantly.Using QR codes + blockchain + cloud storage, AgroLedger allows anyone to trace the origin and details of agricultural products in seconds.


🚀 Key Features
	•Farmer-Friendly Input System
    Farmers can easily enter crop details like name, type, location, and harvest date.
	•Unique Batch ID Generation
    Each crop batch gets a unique ID for identification.
	•Blockchain Integration
    Ensures data is secure, tamper-proof, and trustworthy.
	•Cloud Backup (JSONBin)
    Works even without blockchain access.
	•QR Code Verification
    Buyers can scan a QR code to instantly view product details.
	•Smart Verification System
	  •Checks blockchain first
	  •	Falls back to cloud if needed
	  •	Displays verification status clearly



🧠 How It Works
	1.	Farmer Adds Data
	  •	Inputs crop details via a simple interface
	2.	Batch Creation
	  •	System generates a unique Batch ID
	3.	Storage Decision
	  •	If wallet connected → Blockchain
	  •	Else → Cloud storage
	4.	QR Code Generation
  	•	A QR code is created for that batch
	5.	Buyer Verification
  	•	Buyer scans QR code
  	•	System retrieves and verifies data
	6.	Result Display
  	•	Shows product details + verification status
